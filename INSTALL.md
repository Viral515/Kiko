# Установка Kiko Voice Assistant

## Быстрая установка

```bash
# 1. Установите зависимости
python install_dependencies.py

# 2. Запустите ассистента
# Варианты запуска:
# - start_kiko.bat (с консолью)
# - start_kiko_hidden.vbs (скрытый запуск)
# - python src/main.py (ручной запуск)
```

## Ручная установка

```bash
# 1. Установите основные зависимости
pip install -r requirements.txt

# 2. Установите PyAudio
pip install pipwin
pipwin install pyaudio

# 3. Запустите ассистента
# Варианты запуска:
# - start_kiko.bat (с консолью)
# - start_kiko_hidden.vbs (скрытый запуск)
# - python src/main.py (ручной запуск)
```

## Проверка установки

После установки в трее Windows появится иконка (синий круг с буквой "K").

- **Правая кнопка мыши** → "Запустить ассистента"
- Скажите **"джарвис"** для активации
- Протестируйте команды: **"браузер"**, **"выключи звук"**, **"спасибо"**

## Устранение проблем

### Проблемы с PyAudio
```bash
# Альтернативный способ установки
pip install pyaudio
```

### Проблемы с микрофоном
- Проверьте права доступа к микрофону в Windows
- Убедитесь, что микрофон работает в других приложениях

### Проблемы с звуком
- Проверьте настройки громкости
- Убедитесь, что динамики подключены
